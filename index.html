LanCache DNS Configuration

Configure your devices to use the LanCache DNS server for optimized game downloads

Windows Configuration
1Open Network Settings

Go to Control Panel > Network and Internet > Network and Sharing Center

2Change Adapter Settings

Click on "Change adapter settings" on the left sidebar

3Properties

Right-click your network connection and select "Properties"

4DNS Settings

Select "Internet Protocol Version 4 (TCP/IPv4)" and click "Properties"

5Use Custom DNS

Select "Use the following DNS server addresses" and enter:

Preferred DNS server: 10.160.23.177
Alternate DNS server: 8.8.8.8
Router Configuration
1Access Router

Open a web browser and enter your router's IP address (usually 192.168.1.1 or 192.168.0.1)

2Login

Enter your router's admin credentials (check router manual if unknown)

3Find DNS Settings

Navigate to Internet or WAN settings, then look for DNS settings

4Set DNS Server

Enter the LanCache DNS IP as the primary DNS server:

Primary DNS: 10.160.23.177
Secondary DNS: 1.1.1.1
5Save and Reboot

Save your changes and reboot the router for settings to take effect

About LanCache DNS

Your LanCache DNS server (10.160.23.177) provides the following benefits:

  • Faster game downloads by caching content locally
  • Reduced bandwidth usage for frequently downloaded games
  • Improved download speeds for multiple devices on your network
  • Automatic redirection of supported game CDN traffic to your cache server
Tip: After configuring DNS, restart your game clients to ensure they use the cached content.
Test Your Configuration

Verify that your DNS configuration is working correctly:

Windows Command Prompt
nslookup lancache.steamcontent.com

Should return: 10.160.23.177

Linux/Mac Terminal
dig lancache.steamcontent.com

Look for the ANSWER SECTION showing 10.160.23.177
